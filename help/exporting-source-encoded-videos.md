# Exportation de vidéos source et encodées{#exporting-source-and-encoded-videos}

Dans le panneau de navigation, les vidéos sources sont intitulées « Vidéos originales ». Les vidéos encodées sont intitulées « Vidéos ». Vous pouvez exporter des vidéos source et des vidéos originales. Pour exporter des fichiers vidéo, sélectionnez-les dans le panneau de navigation, puis cliquez sur **[!UICONTROL Fichier]** > **[!UICONTROL Exporter]**.

Voir [Exportation d’actifs à partir de Dynamic Media Classic](exporting-assets-from-dmc.md#exporting-assets-from-dmc).
